Daemon Service

The Apilo daemon runs as a background service, providing automatic API optimization without user intervention.

Key Features

  • Background Processing - Runs independently, no manual intervention
  • Resource Efficient - <50MB RAM, <2% CPU
  • IPC Server - HTTP API on localhost:9876
  • Auto-Recovery - Graceful degradation on errors

Daemon Commands

Command Description
apilo daemon start Start the background daemon
apilo daemon stop Stop the daemon gracefully
apilo daemon status Check daemon health and metrics
apilo daemon restart Restart the daemon
apilo daemon logs View daemon logs
